Custom Theme Instructions

The custom theme must be a text file containing a JSON object, with two elements: a name and an array of RGB tuples. Ideally at least 8 colors.

Example:

  
                {
                    "name": "My custom theme",
                    "theme": [
                    [255,255,255],
                    [120,160,190],
                    [34,78,99]
                    ]
                }
